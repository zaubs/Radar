# Radar

The following is some python code and resulting files of meteor echo data detected by the Canadian Meteor Orbit Radar (CMOR). The parse script is used to collect important components of the meteor's velocity, position in the sky and measurment errors that contribute to the signal coming from the detected echo. Restricitons are defined so that clean signals have a certain threshold of agreement based on their measured values of speed or position reltive to other meteors seen by CMOR.
